Je suis l'ancetre de toutes les boite à l'interieur de moi. De plus, je suis positionné en relatif. Qu'est-ce qui serai différent si je suis positionné en statique?
la possition de cette boite est statique. Il est positionné par rapport à son parent. ( #parent-container ). cette element est non positionné.
Ce élément positionné absolument s'est échappé de son élément parent statique.
Il est maintenant positionné par rapport à son ancêtre relatif le plus proche.
La position relative signifie que l'élément est positionné par rapport à lui-même. Sans utiliser les propriétés top, right, bottom ou left, il se comporte comme un élément statique. Essayez d'ajouter des propriétés pour déplacer cet élément!
Ce élément positionné absolument est positionné par rapport à son ancêtre relatif.
le positionnement fixe signifie que je suis positionné par rapport à la fenêtre du navigateur, c'est-à-dire la fenêtre d'affichage. Même si vous faites défiler, je serai là. PS - informations supplémentaires dans le HTML
